### 核心创新：轮换风险检测器

```python
class RotationDetector:
    def calculate_risk(self, team, opponent, group_table):
        risk = 0
        if already_qualified(team):     risk += 30
        if is_weakest_opponent(opponent): risk += 20
        if key_player_injuries(team):   risk += 25
        if next_is_crucial(next_match): risk += 25
        return ("HIGH" if risk > 60 else "MEDIUM" if risk > 30 else "LOW", risk)
`

这是今天预测最大的教训——系统化评估强队轮换意愿。

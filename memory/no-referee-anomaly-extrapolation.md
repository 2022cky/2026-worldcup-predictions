# 铁律17: 禁止将裁判异常比赛外推

> 来源: 7月10日法国vs摩洛哥预测分析。CLAUDE.md v22 新增。

## 核心原则

当引用一场过往比赛作为"X类型球队面对Y情况表现差"的证据时，必须先检查那场比赛是否存在**裁判/天气/对手策略**等异常因素。一场被裁判摧毁的比赛不是战术分析的合法样本。

## 经典案例

**法国 1-0 巴拉圭 (2026 R16)**: 市场广泛将法国仅靠点球取胜(0.58npxG)解读为"法国破大巴困难"。实际检查:
- 裁判Tantashev完全失控: 巴拉圭28年首次世界杯0黄牌(肘击/掌掴/踢人全不吹)
- 法国反而3黄牌(被犯规方被罚)
- 38.3°C高温(史上最热淘汰赛之一)
- 巴拉圭4场2球/0射正——纯大巴无反击威胁→法国无需冒进
- BBC/Sky/独立报/Express四家同时骂"shameful""embarrassing"

→ 这场比赛的进攻产出**不能**成为评估法国攻击力对大巴的基准。

## S1-S7 强制检查步骤

| 步骤 | 操作 | 说明 |
|:---:|------|------|
| S1 | 识别引用来源 | 标注该比赛的对手/比分/裁判/气温 |
| S2 | 裁判异常检查 | 搜索"[裁判名] referee controversy cards"——检查双方黄牌数 |
| S3 | 对手策略检查 | 纯大巴(0射正)还是会进攻的大巴? |
| S4 | 天气/场地异常 | 极热/暴雨→球员跑动量↓→攻击产出不可外推 |
| S5 | 交叉验证 | 该队对类似级别对手的表现是否一致? |
| S6 | 写对比表 | 防守风格/进攻能力/裁判/气温四维 |
| S7 | 调整权重 | ≥2条异常→从因素表移除锚点→概率上调 |

## 淘汰赛青少年首发规则

**触发**: 2026-07-10 法国vs摩洛哥QF。摩洛哥18岁布阿迪(€8M, 里尔)首发DM → 全场0数据61分钟被换下 → 摩洛哥0-2。赛后写成「淘汰赛不要让18岁小孩首发」→ 被历史反例推翻，需更精确表述。

### 历史反例的共性 — 为什么贝利/姆巴佩/亚马尔不是"普通18岁"

| 维度 | 贝利/姆巴佩/亚马尔 | 布阿迪 |
|------|------|------|
| **赛前身价** | >€100M (或等值) | €8M |
| **赛前队内地位** | 绝对核心/不可替代首发 | 轮换/潜力新秀 |
| **赛前赛季数据** | 顶级联赛核心数据 | 法甲常规轮换 |
| **位置** | 攻击手(FW/RW) | 防守中场 |
| **比赛经验** | 已在大赛(欧冠/联赛)验证 | QF = 职业生涯最大舞台 |

### 正确规则

**青少年(≤19岁)可以在世界杯淘汰赛首发的条件:**

| 条件 | 权重 | 说明 |
|:---:|:---:|------|
| ① 赛前身价 ≥€80M | 高 | 市场已经定价其为顶级球员 |
| ② 赛前已确定为球队**不可替代核心** | 高 | 不是"有潜力"，而是"没有他就没有替代方案" |
| ③ 攻击型位置 (FW/RW/LW/AM) | 中 | 天赋可直接兑现为进球/助攻。防守位置(CB/DM)需要经验，不可被天赋替代 |
| ④ 已在大赛决赛/欧冠淘汰赛级别有首发验证 | 中 | 心理准备已被验证 |

**判定逻辑:**
- 满足 **条件①+②** → 可以在淘汰赛首发（如姆巴佩/亚马尔级别）
- 满足 条件③+④ 但①②未满足 → 可首发但标注风险
- **三项及以上不满足** → 🚨 红色风险 — 标注为对方攻击方向

### 布阿迪的判定
- 条件①: €8M ❌
- 条件②: 非摩洛哥核心 ❌ (阿姆拉巴特才是DM首选)
- 条件③: 防守中场 ❌
- 条件④: 无大赛淘汰赛经验 ❌
- **判定: 4项均不满足 → 首发 = 红色风险**

**Why:** 世界杯QF压力是俱乐部比赛的约3倍。18岁球员的心理和身体准备需要赛前已被精英级别验证。贝利/姆巴佩/亚马尔不是"普通天才"——他们是各自时代的Top 3球员。普通有天赋的18岁和"百年一遇的18岁"之间有一个量级差距。

**How to apply:** 
1. 每次淘汰赛预测 → 检查所有首发中≤19岁球员
2. 跑4项条件筛查
3. 不满足条件 → 在因素表中标注为对手攻击方向：如「奥利塞(€150M)将对位18岁布阿迪(€8M, 4条件均不满足)→法国中路★★★」
4. 在教练博弈中增加「青少年首发 vs 对手相应位置超巨」的对位质量评估

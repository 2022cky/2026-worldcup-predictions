## 关键教训积存
- 均势比赛(<0.25评分差)是模型核心盲区 → 需裁判维度加倍
- 射门数≠攻击力 (土耳其30射1球)
- 核心缺阵影响是非线性的 (唯一核心 vs 多主力)
- 非洲球队在美洲超预期 + 亚洲球队美洲困境
- 英格兰"第二场小组赛综合征"是50年规律
- 大巴成功条件: 排名差>30 + 第二轮 + 弱队只要1分

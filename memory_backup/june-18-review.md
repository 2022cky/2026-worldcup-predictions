## 核心发现: v7过度修正

v7成功修正了葡萄牙vs刚果(错→对), 但把乌兹别克vs哥伦比亚从v6的对→v7的错。

## 核心教训

### 1. 葡萄牙1-1刚果(金) — v7三大新维度全部命中
刚果52年重返世界杯 + 5-3-2大巴 + C罗灾难状态 → 1-1逼平葡萄牙。
v7新增的R6(历史重返)、R7(大巴阵型)、R9(核心灾难)全部触发。

### 2. 英格兰4-2克罗地亚 — 进球通胀被低估
双方顶级射手对决导致进球大战。模型预测2-1偏保守。
R11(进球通胀+12%)应更激进应用。

### 3. 加纳1-0巴拿马 — 帕尔特伊缺阵的"非线性崩塌"
帕尔特伊签证被拒 → 加纳全队90分钟0射门 → 补时绝杀。
模型之前将核心缺阵视为线性扣分(-12%), 实际是体系崩溃。
v8新增R12: 唯一世界级核心缺阵 → 全队进球×0.5

### 4. 乌兹别克斯坦1-3哥伦比亚 — v7的R6过度修正

原因: v7把"首次参赛"给了和"52年重返"一样的+8%平局加成。
v9需新增R6b: 首次参赛加成 = 重返加成的40%(平局+3%, 非+8%)。

## 核心发现

### 1. 捷克1-1南非 — R12规则误触发

v8预测捷克2-0(方向错)。根因: R12(全队进球×0.5)被南非3人停赛触发，但南非缺的不是"唯一世界级核心"，有替代方案。
- **纠正**: R12触发条件收紧为"球队唯一世界级核心缺阵"，不适用于多主力缺阵
- **对比**: 6/18加纳(帕尔特伊=唯一核心→体系崩溃 0射门) vs 南非(多主力→仍有深度→62%控球)

### 2. 墨西哥1-0韩国 — v8四重错误

**预测韩国2-1胜，实际墨西哥1-0胜，方向完全相反**

错误诊断:
1. **高估蒙特斯停赛影响** — 阿尔瓦雷斯客串CB完美填补(R12再次误触发)
2. **高估韩国"下半场爆发力"** — 韩国全场0射正，逆转捷克≠对所有对手都下半场爆发
3. **低估主场高原加成** — 中北美主场(高原+观众+裁判)真实加成>模型估计
4. **高估韩国个体vs低估墨西哥体系** — 阿吉雷防守组织>孙兴慜+李刚仁个人能力

韩国全场0射正 — 世界杯罕见数据，孙兴慜被完全锁死

### 3. 瑞士4-1波黑 — 下半场爆发模式

0-0坚持74分钟，替补奇兵(曼赞比20岁)登场首触球破门改变一切。波黑红牌(80')后崩盘连丢3球。
- **规律**: 红牌方下半场丢球通胀模式继续

### 4. 加拿大6-0卡塔尔 — 亚洲球队美洲困境

加拿大世界杯历史首胜(前2届0胜1球)。大卫帽子戏法=世界杯第56帽，CONCACAF第2帽。卡塔尔2红牌=彻底崩溃。
- **规律**: 亚洲球队(除韩国外)在美洲世界杯表现糟糕，卡塔尔加入中国/沙特行列

## 系统性规律

1. **下半场决定论**: 4场14球中10球在下半场(71%)
2. **红牌=进球通胀**: 红牌方合计丢10球
3. **非洲韧性**: 南非残阵平捷克(刚果平葡萄牙/摩洛哥平巴西 — 非洲不败率高)
4. **亚洲美洲困境**: 卡塔尔0-6/韩国0射正(仅日本表现正常)
5. **主场优势**: 墨西哥/美国/加拿大主场全胜
6. **门将失误主题**: 金承奎脱手(加入卡塔尔/波黑门将失误行列)

## v8→v9模型修正

1. **R12触发条件收紧**: 仅"唯一世界级核心缺阵"触发×0.5，多主力缺阵用×0.8
2. **取消"下半场爆发力"独立加权**: 韩国逆转捷克≠对所有对手都下半场爆发，降为参考因子
3. **主场优势上调**: 中北美主场从×1.1上调至×1.2
4. **非洲球队基础战力上调**: +10-15% (南非/加纳/摩洛哥持续超预期)
